6. GitHubで公開する準備

GitHubに公開するための便利なツールGit Bashをインストールします。Windowsの人はこれでコマンド入力することで簡単にGitHubへ公開することができます。

7.安全な通信の確立

公開鍵/秘密鍵を作っておくことで安全な通信を確立できます。 Git Bashでコマンドを入力して安全に通信できる環境を整えよう！

$ ssh-keygen -t rsa -b 4096

GitHubの自分のアカウントのsettingsからSSH and GPG keysにとんで鍵を覚えさせて準備万端です。

8.やっと公開へ

Git BashでGitHubに公開する。

$ git clone git@github.com:[ユーザー名]/[ユーザー名].github.io.git

これでclone完了。 ディレクトリを移動して

$ cd [ユーザー名].github.io/

addする。

$ git add .

次にcommitして

$ git commit -m "first commit"

下のような返答がきたら

*** Please tell me who you are.

Run

  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"

to set your account's default identity.
Omit --global to set the identity only in this repository.

「”」内をGitHubで登録したメールアドレスとユーザー名にしてコマンドを打つ。

最後に反映するためにgit push。

$ git push https://github.com/[ユーザー名]/[ユーザー名].github.io.git master:master

これで公開！
